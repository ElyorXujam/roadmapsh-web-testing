## "Mark All as Done" bulk action fails to update roadmap node statuses or progress

## Summary
The "Mark All as Done" bulk action on a roadmap does not apply any status changes — all topics remain unmarked after the action is triggered.

## Environment
- Browser: Chrome 125.0
- Os: Windows 11
- Account Type: Registered User

## Steps to Reproduce
1. Go to any roadmap page (e.g., `https://roadmap.sh/roadmaps/frontend`)
2. Ensure you are logged in
3. Locate and click the "Mark All as Done" option
4. Observe the state of the roadmap nodes after the action completes

## Expected Behavior
All topics on the roadmap should be marked as "Done" and the overall progress percentage should update to 100%.

## Actual Behavior
No topics are marked after the action is triggered. The roadmap nodes remain in their previous state and progress is unchanged.
