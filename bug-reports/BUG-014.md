## "Skipped" topics are incorrectly counted toward progress percentage, inflating the roadmap completion rate

## Summary
Topics marked as "Skipped" are incorrectly counted toward the roadmap progress percentage, causing the completion rate to be inflated.

## Environment
- Browser: Chrome 125.0
- Os: Windows 11
- Account Type: Registered User

## Steps to Reproduce
1. Go to any roadmap page (e.g., `https://roadmap.sh/roadmaps/frontend`) while logged in
2. Note the current progress percentage
3. Right-click several nodes and mark them as "Skipped"
4. Observe the updated progress percentage after each mark

## Expected Behavior
Topics marked as "Skipped" should not be counted as completed. The progress percentage should reflect only topics explicitly marked as "Done."

## Actual Behavior
The progress percentage increases when topics are marked as "Skipped," as if they were marked "Done." This inflates the displayed completion rate inaccurately.
