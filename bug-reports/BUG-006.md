## Search results fail to visually highlight matched query terms within result titles and descriptions

## Summary
Search results do not highlight the matched query term within result items, making it harder for users to visually confirm why a result was returned.

## Environment
- Browser: Chrome 125.0
- Os: Windows 11
- Account Type: Registered User

## Steps to Reproduce
1. Go to `https://roadmap.sh`
2. Click the search bar and type "Python"
3. Press Enter or wait for results to load
4. Inspect each result item for any highlighted or bolded instance of the word "Python"

## Expected Behavior
The search term "Python" should be visually highlighted (e.g., bolded or marked with a different color) wherever it appears within each result item's title or description.

## Actual Behavior
No highlighting is applied. The word "Python" appears in plain text within result items, identical in style to surrounding content.
