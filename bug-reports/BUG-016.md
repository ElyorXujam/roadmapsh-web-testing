## Search bar fails to display autocomplete suggestions or dropdown while typing

## Summary
The search bar does not show autocomplete suggestions or a dropdown while typing, leaving users with no real-time feedback as they enter a query.

## Environment
- Browser: Chrome 125.0
- Os: Windows 11
- Account Type: Registered User

## Steps to Reproduce
1. Go to `https://roadmap.sh`
2. Click the search bar in the top navigation to focus it
3. Begin typing "Dev" slowly
4. Observe whether any suggestion dropdown appears below the input

## Expected Behavior
A dropdown with matching suggestions should appear as the user types, providing real-time search assistance.

## Actual Behavior
No autocomplete dropdown appears at any point while typing. The search bar provides no suggestions or live feedback.
